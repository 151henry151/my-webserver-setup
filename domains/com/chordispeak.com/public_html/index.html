About ChordiSpeak

ChordiSpeak is an innovative cross-platform mobile application that uses AI to generate vocal tracks that speak chord names in the original singer's voice. Available on both iOS and Android, our apps are currently in development and will soon be available on the App Store and Google Play Store. Upload any audio file, and our AI will analyze the chord progression and create a vocal track that speaks each chord name using the original vocal characteristics.

How It Works

Processing Pipeline

  1. Audio Preparation: File validation and format optimization
  2. Vocal Separation: AI-powered vocal isolation (10-15 minutes)
  3. Voice Sample Extraction: Characteristic voice pattern analysis
  4. Chord Detection: Advanced chord recognition (2-3 minutes)
  5. TTS Synthesis: Voice cloning for each chord (15-18 seconds per chord)
  6. Audio Mixing: Final track assembly and optimization

Technical Architecture

Models & Data Structures

  • UploadResponse: API response handling for file uploads
  • TaskStatus: Real-time processing status and progress tracking
  • ChordData & Chord: Comprehensive chord progression data structures
  • HealthResponse: Server health and connectivity information

API Layer

  • RESTful API client for seamless backend communication
  • Multipart/form-data file upload with progress tracking
  • Real-time status polling and update mechanisms
  • Comprehensive error handling and response parsing

UI Components

  • AudioFilePicker: Native document picker for audio file selection
  • ProcessingProgressView: Animated progress display with step indicators
  • ChordProgressionView: Interactive timeline visualization of chords
  • ContentView: Modern SwiftUI-based main interface

API Endpoints

  • GET /health - Server health check
  • POST /upload - Upload audio file
  • GET /status/<task_id> - Get processing status
  • GET /download/<task_id> - Download processed audio
  • GET /chords/<task_id> - Get chord progression data
  • POST /cancel/<task_id> - Cancel processing task
